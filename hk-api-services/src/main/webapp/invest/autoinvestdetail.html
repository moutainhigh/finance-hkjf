1.设置自动投资规则后，系统会根据设置时间的先后顺序进行排队；

2.当平台发布新的项目时，如果符合自动投资设置，系统将按设置的排队顺序自动投资;

3.自动投资成功后，自动投资功能设置仍然有效直至用户手动退出(禁用、删除);

4.当账户可用余额小于最低起投金额100时，系统不会触发自动投资功能；

5.只可设置一种投资方案，可以随时编辑该方案，但每次编辑会重新排队；

6.账户余额充足时，手动投资和自动投资可同时使用；

7.当不需要自动投资时，需要手动关闭自动投资功能或选择删除自动投资方案；

8.当系统有多个项目满足其设置的投资方案时，系统会根据投资方案中设置的优先级进行选择，并会将账户中所持有的投资红包、加息券进行最优组合后进行投资；

9。当有符合条件的项目出现时，优先选择账户余额一次性全部投资，不能一次性全部投资的系统将自动进行账户余额的分散投资，投资金额大的优先使用加息券和红包；

10.新手标和爆款标不能使用加息券和红包。
