什么是自动投资
自动投资，是为了方便那些没有时间，逐个查看筛选借款列表，并进行投资的借出者，通过预先设定投资规则后，在无需登录的情况下系统按设定自动完成投资的交易流程。
投资人可根据自己的风险偏好、投资习惯在自动投资功能中设置条件，设置成功并开启此功能后，若有符合相应条件的借款出现，系统将根据用户预先设定的自动投资方案，自动完成投资。
智能投资 省力省心
用户可根据自身资金配置需求，自定义「自动投资」方案。比如标的类型、利率、期限、还款方式及投资方案有效期等进行个性化设置，设置启用后，一旦平台发布的借款标在自定义条件设置范围内，系统将会根据用户「自动投资」设置的先后顺序予以代为投资，真正实现投资智能化，让用户省时省力又省心。
资金不站岗 收益最大化
用户开启「自动投资」功能后，其账户一旦出现多余现金余额，系统便根据用户设置的投资条件，优先选择符合自动投资范围内的标的进行投资，从而更好地解决了用户的资金闲置问题，实现了资金收益最大化。
设置灵活 体验更佳
用户在设置「自动投资」时，可根据个人资金需求自由设置预留金额，且在方案提交后仍可随时更改自定义条件，从而确保用户资金配置的灵活性,令自动投资管理更趋人性化。
