1、协议主体及效力

《鸿坤金服自动投资授权书》（以下称"本协议"）的双方主体为鸿坤金服网站平台（域名：www.hongkunj|nfu.com）的运营商霍尔果斯鸿坤理想科技发展有限公司(以下简称的"鸿坤金服"）与鸿坤金服注册会员。
本协议经鸿坤金服注册会员通过在鸿坤金服设置自动投资条件成功开启自动投资功能、确认已阅读并同意签署本协议后立即生效。


2、自动投资释义

自动投资，是指符合条件的鸿坤金服注册会员通过在鸿坤金服投资系统内设置个人自动投资的投资条件并启用该自动投资功能的，即为全权授权鸿坤金服根据注册会员设置的投资条件及鸿坤金服规定的自动投资规则，通过运行自动投资系统帮助已启用自动投资功能的注册会员完成投资。


3、申请条件

已通过鸿坤金服会员认证并在有效期内的注册会员，均可在鸿坤金服系统设置自动投资条件并开启自动投资功能。

4、自动投资规则

已在《鸿坤金服自动投资规则》中约定的规则与程序，为有效开启自动投资功能的会员提供自动投资服务。鸿坤金服会员必须在签署本协议之前认真阅读并充分理解鸿坤金服已公示的《鸿坤金服投资自动投资规则》。鸿坤金服会员一旦签署本协议即表示并完全接受《鸿坤金服自动投资规则》。与本协议具有同等法律效力。


5、自动投资功能的启用生效

符合条件的鸿坤金服注册会员，按照鸿坤金服自动投资的规定设置自动投资条件后，只要选择"我已阅读并同意签署鸿坤金服自动投资协议"并按照鸿坤金服自动投资规定的程序开启自动投资功能的，鸿坤金服注册会员的行为即表示同意并签署了本协议。自动投资功能设置成功后，该注册会员立即享受鸿坤金服提供的自动投资服务。

6、自动投资功能的终止

鸿坤金服注册会员成功设置并启用鸿坤金服自动投资功能后，发生以下情形之一的，鸿坤金服有权终止该注册会员的自动投资功能：
〔1〕鸿坤金服注册会员的服务功能终止的；
〔2〉会员因违反鸿坤金服公布的各项规则及所签署的各类协议约定的，鸿坤金服认为应当终止该会员的自动投资服务的；
〔3〕其它鸿坤金服认为应当终止注册会员的自动投资功能的情形。
〔4〕鸿坤金服注册会员自行终止自动投资功能的。

8、规则修改

鸿坤金服有权不时按照实际需求，修改《鸿坤金服自动投资规则》，以准确地反映鸿坤金服的会员权益。《鸿坤金服自动投资规则》的所有修改，鸿坤金服会在更改之日在网站公示并即时生效。鸿坤金服会员应不时地注意《鸿坤金服自动投资规则》的变更如不同意规则变更的内容，应立即自己操作取消自动投资功能，否则本协议继续有效。
